# OEasyPool

# 关于错误返回值

以Window的返回值为标准
可以通过VS中的错误查找工具进行错误码查询

错误查找工具的打开方式

 Visual Studio 2013 --> 工具(Alt + T) --> 错误查找(K)
 

# 关于文件编码

该份源码使用的是 Visual Studio, 故编码使用的是 GB2312.


# 关于调用
